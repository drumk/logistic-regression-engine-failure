# Логистическая регрессия - классификация неисправностей двигателей

Классификация двигателей на исправные и неисправные на основе данных о вибрации и неравномерности вращения с использованием логистической регрессии.

## Задача
Разработать систему искусственного интеллекта, классифицирующую состояние двигателя на основе показателей вибрации и неравномерности вращения.

## Данные
Файл `ex2data1.txt`:
- Первый столбец: уровень вибрации
- Второй столбец: уровень неравномерности вращения  
- Третий столбец: метка класса (0 - исправен, 1 - неисправен)

## Математическая модель

### Логистическая регрессия
hθ(x) = σ(θ₀ + θ₁x₁ + θ₂x₂ + θ₃x₁x₂)
где:
- hθ(x) - вероятность принадлежности к классу 1
- σ(z) = 1/(1 + e⁻ᶻ) - сигмоидная функция
- θ₀, θ₁, θ₂, θ₃ - параметры модели
- x₁, x₂ - нормализованные признаки

### Функция стоимости
J(θ) = -1/m * [y⋅log(h) + (1-y)⋅log(1-h)]
где m - размер обучающей выборки

### Градиентный спуск
θⱼ = θⱼ - α * ∂/∂θⱼ J(θ)
